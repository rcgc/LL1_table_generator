# LL1_table_generator
## Abstract


## How to use it
Execute linux commands in the following order:
- `g++ generador_tablas_LL1_linux.cpp` : compiles .cpp file
- `./a.out` : runs the compiled file
- paste input

### Example
#### Input
5 4 <br>
goal -> A <br>
A -> ( A ) <br>
A -> two <br>
two -> a <br>
two -> b <br>
( ( a ) ) <br>
( a ) ) <br>
( ( ( ( ( b ) ) ) ) ) <br>
( ( ( ( ( a b ) ) ) ) ) <br>
